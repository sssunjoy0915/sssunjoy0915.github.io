2019.02 | 二月为什么这么短

发表于 2019-02-28

伴随着连绵的阴雨天，2月也接近尾声。逼自己多写东西，逼自己写月度总结，所以现在正坐在KFC敲着键盘。

刚回到工作岗位的我状态并不是太好，比如前两天其实也没有怎么加大工作强度，却感觉全身乏力，提不起劲，让人感叹难道真是我老了？前几天上级找我谈话，也聊到关于「职业规划」的问题，我回答“我没有想太多”。想了想，毕业快两年了，对于个人的职业或者说事业，似乎还是「走一步算一步」的心态，反正「计划赶不上变化」，“先干了再说吧”——大部分时候是这么想的。有过痛苦的时候，也不止一次想过「辞职」，不过我好像不算是「冒险」的人，如果「裸辞」，我想我会有很大的压力，更何况，现在好像也没有「跳槽」的资本。上级说，她以前是很纠结的人，做一件事情前会考虑很多，一句话怎么讲会设想好多种版本，哈哈哈难道是摩羯座的性格特质嘛。我觉得有的时候我和她一样，做事不够果断，会顾虑很多东西，尤其是重要的事情，一般不会轻易做决定。当然，也有一些冲动和立马行动的时候。

回想整个2月，好像也没有什么特别的事情发生。生活简单而重复，不管怎么过还是会过去。接下来的3月，应该又会全身心投入到工作中，为了达成目标！虽然天气不好，我也会丧丧的，但是希望自己还是能努力撑下去。希望过得更开心更自由。希望能遇到好看的单身小哥哥（哈哈哈）。

最近在追日剧《卖房子的女人的逆袭》，主要是给自己打鸡血。

最后一天更新

昨天见到了好久没见到的小哥哥，心情一下子就明朗起来了呢。

今天一上午都在开会，得知我司员工平均年龄是26岁，还蛮惊讶的。这么说来我已经在平均年龄段了嘛？？？

其实整个2月份都挺不适合上班的，总之我自己的状态很不好。可能是有点厌倦了吧，也可能情绪没有管理好。

今天居然有人评价我「热情」，我觉得是还蛮少听到的评价呢。当然，我热情的时候会很热情呀，可能是真的喜欢，也可能是有些时候和情况下我必须要热情。其实大部分时间我还是非常「冷」和「傲」和「不主动」的人。

每个人在工作的时候都会有烦恼和压力吧，虽然我一直觉得「有痛苦才有成长」，但是谁也不想经历诸多苦难。谁不想一帆风顺万事胜意地过日子呢？

我有时候蛮厌烦现在的工作的，虽然也有开心的很有成就感的时候。 所以怎么样保持对工作的热情呢？ 我觉得自己并不是一个事业心很强的人。不过如果有一个目标需要完成，我应该会竭尽所能去做，可以说是一个目标意识很强的人。

好了， 希望接下来的3月份要打起精神，找回工作状态，努力努力。

FLAG时间
  • 读完至少3本书《给孩子的科幻》等
  • 定期锻炼身体，争取长胖五斤
  • 更主动更多和客户沟通
  • 每天坚持护肤
  • 早起五分钟留时间化妆
  • 坚持好好吃早餐
  • 坚持自己做便当吃
  • 每天记账

（悄咪咪说个愿望：希望能和好看的小哥哥谈恋爱。嘻嘻）
