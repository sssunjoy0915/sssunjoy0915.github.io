吾日三省吾身|190322

发表于 2019-03-22

又有半个多月没有写东西，本来其实也懒得写，也没有太多时间。但是最近心里还是憋了挺多情绪的，我一般也不会找人去倾诉，所以就还是写下来吧，权当给自己梳理心情了。

我算是一个挺容易满足的人吧，就有一些些小东西就能感到很开心的，大概也就是所谓的“小确幸”。比如看到盛开的花、发芽的树，每天好吃的美食，晚霞、落日等等。当然，这样的我也容易因为天气或环境的变化而受到影响，比如下雨天或阴天的时候心情就会低落很多。总而言之，可能就是「钝感力」比较低吧。

最近生活的一些小变化。
  1. 入手了小电动车来代步，所以上下班不用挤公交车了，太自由啦，蛮开心的。
  2. 争取每天自己带饭吃，毕竟外面的东西都不咋样而且还贵（真的，毕业后才懂得学校食堂是多么实惠啊）。
  3. 地铁上尽量逼自己看书，或者听喜马拉雅学点东西，准备好好利用碎片化时间。
工作上的一些不开心。
  1. 毕业快满两年了，正式到现在的部门也快大半年了，说实话，感觉还不是很喜欢或者说适合现在的工作吧。尤其是最近，和主管的关系也不是很好。部门里的大家都是自顾自的，虽然表面上一团和气，但背地里暗潮汹涌。
  2. 为什么主管赢不得大家的敬仰呢？我想了想，有几个原因：年纪太轻了，比我们也就大一岁甚至是同级（当然不是说公司看年龄来决定谁晋升）；能力也就挺一般的，总感觉规划不够清晰，执行时又总是反反复复，变动太多，让人白干好多事又浪费时间；和我一样，都属于不怎么主动交流的人，所以导致工作上的一些事情部门内的人都不知道，后面才恍然大悟，感觉莫名其妙的。
  3. 这份工作给我带来多少成长？这个问题其实也是自己最近一直在思索的。当然这个一方面取决于工作本身的价值以及所需的技能，另一方面还是得看自己的态度和努力。而目前的我感觉大概这两方面都不咋样。
  4. 综上，我挺想换工作的，虽然年初的时候给自己定下了年薪20w的目标，要是现在辞职估计就达不到了。而且现在的工作也真不容易找啊。心塞塞。
个人的生活。

其实我有时候也蛮讨厌自己的性格的。我真的是一个好不主动好不会沟通和表达的人啊。总结一下，就是「情商低」啊。这一点不管是在工作上还是感情上，都挺惨的。之前不是说我有一个喜欢的小哥哥吗。对不住各位了，我到现在也没表白呢，甚至连比较方便的联系方式都还没要到。所以我们一星期大概也就见三次面左右，见面时也说不上几句话。上周感觉他是个性格蛮好的人，但大概也有点羞涩吧，于是我俩就处在原地踏步的阶段。
而另一边，我表姐又提起来我大四时候就介绍过的那个男生，说一定要出来见个面吃顿饭。妈呀这根本都没聊过天的，尴尬极了。我真不是会说话会聊天的人啊。反正就光蒙着头吃日料了。吃完就散伙了，连个联系方式都没留。就是如此随便和草率的「相亲」啊哈哈哈。所以就是我俩都没看上对方呗。就这样呗，随缘。

最近在看的剧和书。
  • 国产剧《都挺好》
  • 书籍《人人都是产品经理2.0》《上帝掷骰子吗？》《世界是部金融史》

好啦，今天来不及写看这些的感受和体会了，先这样吧。
祝看到这里的你很开心哦。
